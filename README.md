ПОРІВНЯННЯ АЛГОРИТМІВ СОРТУВАННЯ

Цей проект порівнює три алгоритми сортування:

1. Insertion Sort (сортування вставками)
2. Merge Sort (сортування злиттям)  
3. Timsort (вбудована функція sorted())


ДОДАТКОВЕ ЗАВДАННЯ:

Файл merge_lists.py містить функцію для об'єднання 
декількох відсортованих списків.

Запустіть файл merge_lists.py:
python merge_lists.py

РЕЗУЛЬТАТИ:

На малих масивах Insertion Sort може бути швидшим.
На великих масивах Merge Sort значно швидший.
Вбудований sorted() завжди найшвидший, бо це оптимізований алгоритм.
